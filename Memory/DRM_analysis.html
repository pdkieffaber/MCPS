Deese–Roediger–McDermott (DRM) paradigm

Studying False Memory

Basic Procedure

  1. Present a list of semantically related words
  2. Omit the most strongly related “critical lure”
  3. Test recall or recognition
  4. Measure how often the lure is falsely remembered

Example

Presented:
bed, dream, tired, wake, snooze, blanket

Not presented:
sleep

What it shows

  • Memory is not a literal replay
  • Meaning-based encoding can drive errors
  • Retrieval can produce vivid but inaccurate recollections

Participants often report the missing lure with high confidence, even though it never appeared.
